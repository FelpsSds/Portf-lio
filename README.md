# Portf-lio

Portfólio pessoal de José Felipe Silva, criado com HTML e CSS.

Este projeto apresenta uma página simples e minimalista com:

- Seção inicial com apresentação pessoal
- Seção "Sobre mim" com informações de formação e experiência
- Contatos diretos para GitHub, LinkedIn e WhatsApp
- Foto de perfil exibida na landing page

## Tecnologias

- HTML
- CSS

## Como executar

1. Abra o arquivo `index.html` no navegador
2. Ou hospede o projeto em um serviço como Vercel

## Deploy

Link de produção no Vercel:

https://portf-lio.vercel.app

> Se este link não corresponder ao seu domínio Vercel, substitua pelo URL correto.
